InstanceMeta

Index

  • instance_traffic

Field

字段 类型 长度 落库 含义
serviceName String 默认70 name 长度配置 serviceNameMaxLength
instanceName String 默认70 - 长度配置 instanceNameMaxLength
serviceId String - service_id 生成规则 base64(serviceName).(normalType 0/1)
properties JSON - properties
- OS Name - - 系统
- hostname - - 主机
- Process No. - - 进程号
- language - - 语言 如 Java
timeBucket long - time_bucket 注册时间 默认Minute YYYYMMDDHHmm
lastPingTimestamp long - last_ping 最近一次心跳检测时间 默认Minute YYYYMMDDHHmm
id long - es id serviceId_base64(instanceName)

Combine

  • 更新 lastPingTimestamp
  • 更新 properties (为空不更新)
  • 更新 timeBucket 保持为首次注册时间

equals&&hashCode

  • serviceId + instanceName

update

  • 支持更新
