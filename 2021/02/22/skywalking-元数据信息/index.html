skywalking系列 | 元数据信息

2021年2月22日

服务信息

表名

srvice_traffic

字段属性

字段 类型 含义
service_group String 服务组名
name String 服务名
node_type integer 节点类型
id String base64(serviceName).(normalType)

聚合&比较规则

  • 不聚合
  • 按 服务名称 + 节点类型 比较

采样&更新规则

  • 不设置采样率
  • 增量插入，非增量不处理

实例信息

表名

instance_traffic

字段属性

字段 类型 含义
name String 实例名称
service_id String 服务ID
properties JSON properties
OS Name 系统
hostname 主机
Process No. 进程号
language 语言 如 Java
time_bucket long 注册时间 默认分钟 YYYYMMDDHHmm
last_ping long 最新心跳时间 默认分钟 YYYYMMDDHHmm
id long serviceId+base64(instanceName)

聚合&比较规则

  • last_ping: 取本次上传
  • properties：不为空，取本次上传
  • time_bucket：取库中数据，保持不变
  • 比较规则：serviceId + name

采样&更新规则

  • 不设置采样率
  • 增量插入，非增量部分更新

端点信息

表名

endpoint_traffic

字段属性

字段 类型 含义
name String 端点名
service_id String 服务ID
time_bucket long 注册时间 默认Minute YYYYMMDDHHmm 首次Span#startTime
id String serviceId+base64(endpointName)

聚合&比较规则

  • 不聚合
  • 对象比较

采样&更新规则

  • 不设置采样率
  • 增量插入，非增量不处理

地址映射

表名

network_address_alias

字段属性

字段 类型 含义
address String ip/hostname:port
represent_service_id String 服务ID
represent_service_instance_id String 实例ID
time_bucket long 注册时间 默认Minute YYYYMMDDHHmm 首次Span#startTime
last_update_time_bucket long 更新时间 默认Minute YYYYMMDDHHmm
id String base64(address)

聚合&比较规则

  • last_update_time_bucket: 取本次上传
  • represent_service_id: 取本次上传
  • represent_service_instance_id: 取本次上传
  • time_bucket：取库中数据，保持不变
  • 比较规则：`address

采样&更新规则

  • 不设置采样率
  • 增量插入，非增量部分更新
