APDEX性能指数

参考：apdex

  • APDEX性能指数（Application Performance Index）是一个国际通用的应用性能计算标准。该标准将用户对应用的使用感受定义为三个等级：
    • 满意（0~T）
    • 可容忍（T~4T）
    • 不满意（大于4T）

  • 一般：取应用的平均响应时间作为计算指标，并将T定义为500毫秒。
Apdex值 颜色 说明
0.75 ≤ Apdex ≤ 1 绿色 表示应用、实例或事务被调用时响应很快，用户体验较满意。
0.3 ≤ Apdex < 0.75 黄色 表示应用、实例或事务被调用时响应较慢，用户体验一般。
0 ≤ Apdex < 0.3 红色 表示应用、实例或事务被调用时响应极慢，用户体验较差。
