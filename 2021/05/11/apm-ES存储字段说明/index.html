ES索引模板

ES索引表

  • 明细：apm_record-yyyyMMdd

注：按天存储

数据类型说明

名称 含义
segment_record Segment明细
trans_slow_record 慢事务明细
db_slow_record 慢sql明细
browser_error_log_record 浏览器错误日志明细
alarm_message_record 告警明细
thread_snapshot_record 性能剖析明细
alarm_message_record 告警明细
trace_metrics 指标明细
slow_trans_metrics 慢事务性能指标
db_metrics 数据库性能指标
slow_page_metrics 慢页面性能指标
browser_pref_metrics 浏览器性能指标

基础字段

字段 类型 含义
service string 服务名

tags string 自定义标签
