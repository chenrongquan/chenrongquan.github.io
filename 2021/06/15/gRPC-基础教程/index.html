2021年6月15日

基本要求

  • 在proto文件中定义服务
  • 使用 protocol buffer 编译器生成 服务端 和 客户端 代码
  • 使用 Java gRPC API 写一个简单的客户端 和 服务端 的服务

使用gRPC

  1. 可以在proto文件中定义服务
  2. 以gRPC支持的任何语言生成客户端和服务端，可在不同的环境或语言中运行
  3. 使用protocol buffer的优势，包括高效的序列化、简单的IDL、轻松的界面
